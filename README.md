# md-toc-ln

CLI that prints a table of contents with line numbers for a given markdown file or stdin.

## Installation

```bash
bun add md-toc-ln
```

## Usage

```typescript
import { greet } from "md-toc-ln";

console.log(greet("World")); // Hello, World!
```
